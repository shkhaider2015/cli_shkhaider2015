# Usage
<!-- usage -->
```sh-session
$ npm install -g shkhaider2015
$ shkhaider2015 COMMAND
running command...
$ shkhaider2015 (-v|--version|version)
shkhaider2015/0.0.0 win32-x64 node-v14.17.1
$ shkhaider2015 --help [COMMAND]
USAGE
  $ shkhaider2015 COMMAND
...
```
